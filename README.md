# Cube Surfer

A clone of a hyper-casual game Cube Surfer.

The gameplay of the source game can be found [here](https://youtu.be/6YIw4G-2Su4).

> This project is made for educational purposes.

## How to play

Rules:
  - Collect yellow cubes to earn more gems in the end of the level.
  - Avoid red cubes and lava as much as possible.
  - Win when reached the finish.
  - Lose when out of cubes.

Controls:
  - `Touch` the screen and `Swipe Left/Right` to move.
  - Alternatively, you can move with `Keyboard Arrows`.

## Used technologies

- [LeoECS](https://github.com/Leopotam/ecs) - Entity-Component-System
- [DOTween](http://dotween.demigiant.com/) - UI animations.
- [Blockbench](https://www.blockbench.net/) - For simple 3D models (cubes and curved platforms).
